# Program Functionality
This program scans WiFi networks around the user, displays connection information, signal quality, security, and makes it easy for users to monitor, save, and analyze WiFi through an interactive GUI display.

# Key Features
1. Universal WiFi Scan
2. Supports Windows & Linux.
3. Detects connected and available WiFi networks in the vicinity.
4. Auto-detects USB WiFi interfaces.
5. Interactive GUI (PyQt5)
6. User-friendly, easy-to-remember buttons and tables.
7. Dark mode / Light mode.
8. Real-Time Monitoring
9. Automatic scanning every 5 seconds.
10 .Real-time signal chart using matplotlib.
11. Multi-Thread & Progress Bar
12. Scanning runs in the background without freezing the GUI.
13. Displays scan progress.
14. Data Export
15. Saves scan results to HTML, JSON, TXT.
16. Automatically creates folders to organize files.
17. Requests user permission before saving.
18. Desktop Notifications
19. Alerts when new WiFi is detected or signal drops.
20. Portable EXE (Windows)
21. Can be run without installing Python.
